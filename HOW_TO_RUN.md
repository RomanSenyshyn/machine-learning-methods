# Як запустити програму / How to Run

## Крок 1: Встановлення залежностей / Step 1: Install Dependencies

Відкрийте PowerShell або Command Prompt і виконайте одну з команд:

### Варіант 1: Встановлення для користувача (рекомендовано)
```bash
pip install --user numpy pandas matplotlib seaborn scikit-learn
```

### Варіант 2: Встановлення глобально (потрібні права адміністратора)
```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

### Варіант 3: Використання віртуального середовища (найкращий варіант)
```bash
# Створіть віртуальне середовище
python -m venv venv

# Активуйте його
# Для PowerShell:
.\venv\Scripts\Activate.ps1
# Або для Command Prompt:
venv\Scripts\activate.bat

# Встановіть залежності
pip install -r requirements.txt
```

## Крок 2: Запуск програми / Step 2: Run the Program

**Важливо:** Використовуйте прапорець `-u` для коректного виведення:

```bash
python -u iris_classification.py
```

Або просто:
```bash
python iris_classification.py
```

**Примітка:** Якщо ви не бачите вивід, спробуйте `python -u iris_classification.py`

## Що робить програма / What the Program Does

Програма виконає наступні кроки:

1. **Завантажить дані Iris** - класичний набір даних для класифікації
2. **Покаже статистику даних** - перші рядки, інформацію про дані
3. **Створить візуалізацію** - pairplot з розподілом ознак (збережеться як `pairplot.png`)
4. **Розділить дані** - на тренувальну (70%) та тестову (30%) вибірки
5. **Навчить 3 моделі**:
   - Логістична регресія
   - Дерево рішень (з візуалізацією - `decision_tree.png`)
   - K-найближчих сусідів (з графіком впливу k - `knn_accuracy.png`)
6. **Покаже результати** - точність кожної моделі та порівняння

## Примітки / Notes

- Графіки будуть відображатися в окремих вікнах
- Зображення також зберігаються у файли PNG
- Якщо виникають проблеми з правами доступу, використовуйте `--user` або віртуальне середовище
